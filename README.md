# tele-rss-gram

Polls a list of RSS feeds and pushes any new items to a Telegram group

## Deployment

1. Create a venv & install dependencies

    ```python
        python3 -m venv .venv
        source .venv/bin/activate
        pip install -r requirements.txt
    ```

2. Set your configuration

    `nano src/config.yml`
